Click on a button on the left to select a category. From there, you can select a game from the set of buttons which appear in the middle column.
Games have different types, indicated by icons on the buttons. Types include fullscreen, unblocked, scratch, turbowarp, flash archive, and gameboy. Hover over an icon to see its name if you can't tell by looking at it.
Search for games using the search bar above the category list. Games will not be included if the search term does not appear in any of the category name, tags, or game types. Combine multiple keywords by separating them with a comma followed by a space, like this.
Tags for the selected category are shown underneath the game list. Click on one to enter it into the search bar.
Click on the link icon to the left of the tags to embed the selected game into the URL. Then, you can copy the URL to bookmark the selected game or share it with friends.
